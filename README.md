 # SimplyOS

SimplyOS is a windows playable operating system with different apps and os features.

<img width="2560" height="1440" alt="image" src="https://github.com/user-attachments/assets/17a5f07e-c7db-4fae-8c70-c6675aac8795" />

## Features
- Apps
- Taskbar
- Start menu
- Settings
- Desktop pet

## How to download
1. Download the zip file from releases
2. Extract/unzip the .zip file
3. Open the extracted foler and open SimplyOS.exe (application)

If Windows SmartScreen blocks it, click - More info, and then - Run anyway

## Tech stack
- Java
- VS Code
